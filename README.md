# SecretSanta


## Backend Setup
### Setup on linux
```
For Linux:
cd backend
source venv/bin/activate
pip install -r requirements.txt
```

### Setup on windows
```
cd backend
.\winvenv\env\Scripts\activate
pip install -r requirements.txt
```

### Setup on Both
```
python manage.py migrate
```

## Frontend Setup
### Node
```
cd frontend
npm install
```
